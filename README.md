# Demand-Prediction-using-Multiple-Linear-Regression

# Project Name: 
Demand Prediction using Multiple Linear Regression
# Description:
Using multiple Linear Regression to predict demand of a particular product based on multiple factors. Also includes Exploratory Data Analysis and identifying the significant factors, detecting Multicolinearity and checking for Variance Inflation Factor and creating dummy variables for categorical variables.
